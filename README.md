# DRL baseline

目前主要是一些单智能体深度强化学习方法的复现

学生习作，以大神sutton的书和周博磊的课程为线索复现

## 目前已完成的部分包括：

* DQN
* REINFORCE
* REINFORCE with baseline
* one-step AC
* QAC
* QAC with shared network
* PPO2
* DDPG
* TD3
* SAC
* SAC (离散动作)
* A2C
* A3C
